# Gearcraft — front (Vite + React)

Réplica do `whacked_front`: mesma API (`/api` → `http://127.0.0.1:3001`). UI com tema **roxo / violeta** como [gearcraft.gg](https://gearcraft.gg/) (auroras, gradiente “Catalog”, scrollbar, chips ativos).

- `npm install` → `npm run dev` (porta **8081** para não colidir com Whacked na 8080).
- API na pasta `../server`; env em `../.env`.
